### Run Locally

To run **Pixology** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone git@github.com:yassaasalib/pixology.git
```

Windows:

```bash
git clone git@github.com:yassaasalib/pixology.git
```
